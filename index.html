Get Started
Zero-code log capture

Capture every log.
Change zero code.

Server logs from tmux. Browser console from an extension. One YAML config file, one command, and you're done. No SDKs, no wrappers, no instrumentation.

curl install View on GitHub
devlog.yml 
version: 1
project: my-app
logs_dir: ./logs

tmux:
  session: my-app
  windows:
    - name: dev
      panes:
        - name: web
          cmd: npm run dev
          log: server/web.log
        - name: api
          cmd: pnpm --filter api dev
          log: server/api.log

browser:
  native_host: true
  file: browser/console.log
  levels: [error, warn, info, log]
  urls:
    - "http://localhost:*/*"
Features

Set it and forget it

One YAML file declares everything. devlog handles the rest — tmux sessions, browser logs, timestamps, and cleanup.

tmux log capture

Creates sessions, runs dev commands, and pipes stdout/stderr to timestamped log files. Every pane, every process.

Browser console capture

Chrome + Firefox extension streams console.* calls, uncaught errors, and unhandled rejections straight to disk via native messaging.

One YAML config

Commands, panes, log paths, URL filters, log levels — all in a single devlog.yml. Environment variable interpolation built in.

Timestamped or overwrite

Run in timestamped mode for history, or overwrite mode for a clean slate. Auto-cleanup old runs by count or age.

Single binary

Go binary. No runtime, no Node, no Python. Drop it in your PATH and go. Works anywhere tmux runs.

Health check built in

devlog healthcheck verifies tmux, browser extension registration, and native host — before you start.

Install

One command to start

Pick your install method. Mac, Linux, WSL — anywhere tmux runs.

terminal 
# Quick install
curl -fsSL https://raw.githubusercontent.com/jellydn/devlog/main/install.sh | sh

# Or specify version and directory
VERSION=v0.1.0 INSTALL_DIR=~/.local/bin \
  curl -fsSL https://raw.githubusercontent.com/jellydn/devlog/main/install.sh | sh
terminal 
# Requires Go 1.25+
go install github.com/jellydn/devlog/cmd/devlog@latest
go install github.com/jellydn/devlog/cmd/devlog-host@latest

Then run devlog init to create your config, devlog healthcheck to verify, and devlog up to start capturing.
